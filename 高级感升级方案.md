# 杂研 · 高级感升级设计方案

> 参考 5 个海外站点（glukhovsky / seunghyuk / lemansclassic / obys / eleos）后，为「杂研」（报纸学术风博客）整理出 **5 套可落地方案** + **1 组横向通用细节**。每套方案按"整体框架 / 动态效果 / 排版"三维度给具体改动与工期。

---

## 一、5 个参考站的核心技法对照

| # | 站点 | 一句话印象 | 可偷的核心技法 |
|---|------|-----------|----------------|
| 1 | glukhovsky.com | 编辑专栏式作家主页 | 三栏 + **竖排类目导航（带计数）** + 巨字 hero + 章节竖排序号 |
| 2 | seunghyuk.com | 极简辐射式信息架构 | 中央 hub + 辐射线 + 噪点底纹 + 顶部 SOUND ON/OFF 这种小心机 |
| 3 | lemansclassic.richardmille.com | 戏剧冲击型品牌页 | 强对比色块（黄/黑/白）+ **横向 marquee 大字** + 大字剪切溢出 |
| 4 | experiment.obys.agency | 沉浸式 3D 作品展厅 | 巨型标题换行 + **3D 浮卡阵列**（perspective 旋转的卡片） |

## 二、5 套可选设计方案

### 方案 A · 编辑风升级（Glukhovsky 同源）
> 最低成本，最贴近你现有的报纸风，能立刻拉高"期刊感"。
- **整体框架**：主页三栏化。左侧：本期目录；中：巨字 hero + 当期头条；右：**竖排类目导航 + 计数**（"文章 142 / 笔记 23 / 谜题 8 / 古董 56"）。
- **动态效果**：右侧计数从 0 滚动到目标值（IntersectionObserver 触发）；竖排类目 hover 时该列变深。
- **排版**：hero 巨字 `Dmitry Glukhovsky` 同款 — 字号拉到 9–14vw；下方小字 "Vol.III · Issue 47 · 2026.07"；副标细衬线斜体。
- **适用页**：首页 `index.html`。
- **工期**：半天。

### 方案 B · 3D 浮卡博物馆（Obys 同源）
> 视觉冲击最大，适合你已有的"线上博物馆"模块。
- **整体框架**：博物馆/精选文章列表用 **3D 浮卡阵列**——一组卡片在 perspective 空间里排成弧形，鼠标拖动时整组旋转。
- **动态效果**：鼠标拖动 / 滚轮 / 触摸滑动旋转整组；hover 单卡时该卡浮起、其余变暗；空闲时缓慢自动漂移。
- **排版**：浮卡正面是常规"展品卡"（图 + 标题 + 年代 + 来源），底面是说明文字（旋转 180° 可见）。中央留大空白写巨型标题如 "EXPERIMENT SPACE"。
- **适用页**：`museum.html`、首页精选区。
- **工期**：1–2 天（CSS 3D transform + JS drag，无新依赖）。
- **风险**：移动端退化为 2D 横滑列表；老浏览器降级为静态网格。

### 方案 C · 辐射式主页（Seunghyuk 同源）
> 最具"小而精"气质，适合做一个独立入口页或博物馆/藏品索引。
- **整体框架**：页面中央一个深色方块 hub，**8 条辐射线**（SVG）连到 8 个类目（博物馆/谜题/古董/笔记/公告 等）。
- **动态效果**：hover 某条线时该线发光、端点粒子流动；点击端点展开扇区展示子类；中心 hub 持续极慢自转。
- **排版**：全站极简 — 顶部 `KIM SEUNGHYUK` 同款小型衬线 + 字距拉宽；类目标签小字号大写 + 微微倾斜（15°）模拟手写坐标。
- **适用页**：博物馆索引、藏品总览、新独立入口页 `index-v2.html`。
- **工期**：1 天（SVG + CSS 动画 + JS 交互）。
- **风险**：搜索/SEO 不友好（核心内容靠图形入口），适合做"实验入口"而非主流量入口。

### 方案 D · 戏剧冲击色块（Richard Mille 同源）
> 最容易"出片"——一条横向大字报头带，立刻有杂志感。
- **整体框架**：主页顶部加一条**横向滚动的巨型 marquee 带**（黄底黑字或黑底白字），内容是 "BREAKING · 本期推荐 · 谜题 #47 · 古董上新 · ..."。
- **动态效果**：持续向左滚动；hover 暂停；点击跳转。整条带占屏宽 100%，高度 8–12vh。
- **排版**：巨型无衬线黑体（`Inter Tight 800` 或 `Archivo Black`），字号 8–12vw，被视口剪切溢出（与 Richard Mille 同款）。
- **适用页**：首页 `index.html` 顶部、公告页。
- **工期**：半天。
- **风险**：移动端 marquee 改用 fade 切换避免晕眩。

### 方案 E · 沉浸氛围详情页（Eleos 同源）
> 把"博物馆藏品详情"和"文章详情"升级成氛围页。
- **整体框架**：详情页 hero 用**全屏 50vh 头图**（藏品照片/文章封面），下半屏硬切到深色纯色块（藏青/深绿/米色）。
- **动态效果**：头图慢速视差（内容滚得快、图滚得慢）；切换到色块时有 0.4s 渐隐过渡。
- **排版**：巨字标题叠在头图上（左下，半透明白字 + 文字阴影）；色块内的正文用浅色细衬线。
- **适用页**：`museum.html` 详情区、`article.html`、古董详情。
- **工期**：半天。

## 三、横向通用（不绑特定方案）

这些小细节**任何方案都该加**，是"高级感"的隐形骨架：

| 细节 | 说明 | 工期 |
|------|------|------|
| ① View Transitions 页面切换 | 浏览器原生 API，同源淡入 + 微缩放，无需框架；Chrome/Edge/Safari 17+ 支持，自动降级 | 1h |
| ② 数字滚动计数 | 计数从 0 → 目标值，IntersectionObserver 触发；适合方案 A 的侧栏计数 | 2h |
| ③ 字符逐字显示 | 标题逐字 fade-in，错落 30ms 间隔；适合 hero | 1h |
| ④ 视差 hero | 头图 `background-attachment: fixed` 已被你修复为 scroll，可改用 `transform: translateY(scrollY*0.3)` 实现真视差 | 2h |
| ⑤ 期号 / Vol 标识 | 在 masthead 加 `Vol.III · Issue 47 · 2026.07.22` 强标识，强化编辑身份 | 30min |
| ⑥ 磁性按钮 | 你已有自定义光标，给主 CTA 按钮加"光标靠近放大 1.05" | 1h |
| ⑦ 滚动揭示错落 | 已有 `.reveal-group` 机制，可给文章卡组加 `--reveal-i` 自动错落 | 已落地 |
| ⑧ 数字/装饰字符 | 巨号字（"01" "02"）作章节起始标识，类似杂志章节号 | 1h |

## 四、推荐组合

| 组合 | 包含 | 效果 | 工期 | 适合 |
|------|------|------|------|------|
| 🅰 **保守升级**（最快出效果） | A（编辑风）+ E（氛围详情）+ 通用 ①②⑤ | 纯静态升级，编辑身份拉满 | 1.5 天 | 答辩前赶工 |
| 🅱 **冲击组合**（推荐） | A + B（3D 浮卡）+ D（marquee）+ 通用 ①②③⑤⑥ | 视觉冲击 + 编辑感 + 动态节奏 | 3–4 天 | 有时间打磨 |
| 🅲 **实验入口** | C（辐射式主页）作为 `index-v2.html` 独立入口，主页仍用 A | 双入口，A 主流量 / C 实验室 | 2 天 | 想做"作品集彩蛋" |

## 五、下一步

1. 你从 **🅰 / 🅱 / 🅲** 选一个组合（或单独挑 A–E 任意方案 + 通用清单）；
2. 我**一次性**把代码改完（CSS + JS），给可运行结果；
3. 你浏览器自测，所有方案都遵循 `prefers-reduced-motion` 降级。

> 诚实边界：方案 B 的 3D 浮卡只做 CSS 3D 级别，不上 Three.js/WebGL（无新依赖、无构建）。如需真正 WebGL 浮卡（光照、阴影），那是另一个量级的工作，要单独排期。
